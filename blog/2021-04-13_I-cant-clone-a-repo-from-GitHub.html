Can't clone a repo from GitHub

Sprint 1Tech

Issue

When cloning a repo from GitHub during Sprint 1 Challenge, I kept getting this error message. My username and password are correct, but I still couldn't get it through.

$ git clone https://github.com/xxxxxxxx/ko-wai-koe.git
Cloning into 'ko-wai-koe'...
Username for 'https://github.com': keinskyrocket
Password for 'https://keinskyrocket@github.com':
remote: Invalid username or password.
fatal: Authentication failed for 'https://github.com/xxxxxxxx/ko-wai-koe.git/

Cause

(Probably) Two Factor Authentication (2FA) on GitHub because I enabled it.

Solution

Generate a Personal Access Token on GitHub.

  1. Go to Personal access tokens page
  2. Hit Generate new token button from top right
  3. Type your informative description for the token in Note text field
  4. Tick everything under Select scopes for now
  5. Hit Generate token button at the bottom of the page
  6. Go back to Terminal and hit git clone {#repo-url}
  7. Type your username
  8. Copy and paste the token you just generated from GitHub, and use it instead of your password
